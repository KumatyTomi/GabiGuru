# GabiGuru

Repozytorium projektu **GabiGuru / Gab!Guru Scissors**.

## Cel projektu

Projekt ma porządkować materiały, źródła, katalog produktów, opisy i przyszłą aplikację/stronę dla marki nożyczek groomerskich GabiGuru.

## Status połączenia

To repo jest przypisane jako techniczne repozytorium projektu GabiGuru:

- GitHub: `KumatyTomi/GabiGuru`
- gałąź główna: `main`
- projekt: `GabiGuru`
- zakres: katalog, treści, źródła, assets, przyszła strona/aplikacja

## Źródła startowe projektu

Do dalszego researchu i katalogowania traktuj jako punkty odniesienia:

- oficjalny profil Instagram `@gabiguruscissors`, powiązany z Gabrielą Jędrzejczyk / Gab!Guru Scissors;
- posty i materiały o katalogu „Gab!Guru scissors”;
- wzmianki Facebook/Instagram o katalogu „GabiGuru x Miko”;
- profil/strona **MIKO Ostrzenie Narzędzi**, gdzie pojawiają się informacje o sprzedaży nożyczek groomerskich GabiGuru;
- znany trop katalogowy/modelowy: **Gabi Guru Scissors ZENITH NORI S** — nożyczki proste 6.5, asian style, ceny z widocznych postów około 410 / 360 PLN.

## Następne kroki

1. Dodać materiały źródłowe do `docs/sources.md`.
2. Utworzyć katalog produktów w `docs/catalog.md`.
3. Dodać zdjęcia nożyczek do `assets/product-photos/`.
4. Przygotować strukturę strony/aplikacji w katalogu `app/`.
