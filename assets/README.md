# Assets — GabiGuru

Tu trafią materiały graficzne projektu.

## Plan katalogów

```txt
assets/
├── product-photos/
├── brand/
├── social/
└── raw/
```

## Zasady

- `product-photos/` — zdjęcia modeli nożyczek;
- `brand/` — logo, kolory, identyfikacja wizualna;
- `social/` — grafiki do postów i rolek;
- `raw/` — materiały źródłowe przed obróbką.

Zdjęcia powinny mieć opisowe nazwy plików, np.:

```txt
gabiguru-zenith-nori-s-65-front.jpg
gabiguru-zenith-nori-s-65-detail.jpg
```
